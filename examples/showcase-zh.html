标题与段落

这是正文段落，用于检查字号、行高与段间距。句中可以出现强调文字，也可以出现inline code 这样的行内代码，以及指向本页的链接。连续两段用于确认段落之间的节奏一致。

第二段正文。中英混排时间距应自动留出，例如使用 HTML 与 CSS 渲染，版本号写作 1.0，日期写作 2026-07-18。

小节标题

小节标题没有左侧色条，字重与二级标题一致，用于在长章节内部分层。

列表

无序列表的形态如下：

  • 第一项，检查列表标记的颜色与缩进。
  • 第二项，检查条目之间的间距。
  • 第三项，较长的一条，用于观察折行后的悬挂缩进是否对齐到文字而不是标记。

有序列表的形态如下：

  1. 第一步，确认序号使用等宽数字。
  2. 第二步，确认序号颜色弱于正文。
  3. 第三步，确认与无序列表的节奏一致。

引用与分隔

引用块用于转述决定、原话或外部材料：

引用块的左侧有一条清晰但克制的暖灰竖线，文字使用次级墨色，不使用斜体。这段文字足够长时会自然折行，用于检查行高与左内边距的配合。

分隔线用于章节内的强停顿，前后留白一致：

分隔线之后的段落，确认间距恢复常态。

提示块

普通提示块用于记录决定、说明背景：

决定：轮询刷新

提示块使用沙色底，标题行字重 500，正文与页面对齐。这是提示块内的第二句，确认末段没有多余的底部间距。

警告提示块用于风险与注意事项：

风险：数据延迟

警告块使用警示色底，标题使用警示色文字。整页只允许这一种第二色彩出现。

表格

表格用于并列的结构化信息，数字列使用等宽数字：

项目数值说明
正文宽度680 px阅读列的最大宽度
页面宽度960 px页面外壳的最大宽度
正文字号15 px行高 1.65

代码块

多行代码使用等宽字体，底色为纸面卡片色，超宽内容横向滚动：

body {
  margin: 0;
  background: var(--paper);
  color: var(--ink);
}

控件

按钮状态

按钮承载页面上的操作，在每个状态下都要给出清晰反馈。

开关

开关用于即时生效的设置项。

滑块

滑块用于调节阈值与数值范围，读数随拖动实时更新。

分段控件

分段控件在少量互斥选项之间切换。

表单与反馈

表单校验

输入框在失焦与提交时校验，错误与成功状态就地给出。

对话框

模态对话框打断当前流程，要求用户确认破坏性操作。

轻提示

轻提示用于异步操作的非阻塞反馈，会自动消失。

进度条

进度条展示确定任务的完成比例。

骨架屏

骨架屏在内容到达前占住版式，避免页面跳动。

导航与结构

标签页

标签页在同级视图之间切换，支持方向键操作。

折叠面板

折叠面板把长内容收进可展开的分节。

步骤条

步骤条把多步流程拆成有序的阶段。

卡片列表

卡片列表用于在集合中单选一项。

动效与图示

弹簧动效

相同的移动距离，在缓出与弹簧两种曲线下手感不同。

界面示意

界面示意是静态 SVG，展示布局结构，不提供交互。

关系图

关系图是静态 SVG，分流程图、时序图与状态图三种，不提供交互。
